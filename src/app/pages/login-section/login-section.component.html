POST /user/login-Corporate/merchant

Login API

Authenticate corporate merchants with their mobile number, password, and a 2FA authentication code. Successful authentication returns a JWT token for subsequent API calls.

Request Attributes

Attribute Type Required Description Sample
mobile string Yes Registered mobile number with country code (without + sign) "9182XXXXX94"
password string Yes Account password "test123"
authcode string Yes 6-digit 2FA code from authenticator app "128636"
