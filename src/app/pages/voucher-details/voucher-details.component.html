POST /user/card-list/{orderId}

Get Voucher Details

Retrieve detailed information about a specific voucher by its order ID. Token is required and must be passed in the header.

Request Parameters

Attribute Type Required Description Sample
orderId integer Yes The unique ID of the order to fetch voucher details 12345

Request Attributes

Attribute Type Required Description Sample
search string, null Yes The search term for voucher ID "null", "123"
filterBy string, null Yes The field to filter the search results by "null"
pagination object Yes Pagination details for the results See below

Pagination Object

Attribute Type Description Sample
sortBy string Field to sort results by "id"
descending boolean Sort in descending order false
page integer Current page number 1
rowsPerPage integer Number of items per page 100000000
rowsNumber integer Total number of items available 0
