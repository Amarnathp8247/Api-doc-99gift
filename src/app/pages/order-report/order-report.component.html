Order Reports

Retrieve order reports with filtering and pagination options. Token is required and must be passed in the header.

Request Attributes

Attribute Type Required Description Sample
search string No Search term for filtering orders "ZP1111799"
filterBy string No Field to filter the results by "invoice_id"
date object Yes Date range for filtering orders See below
pagination object Yes Pagination details for the results See below
status integer Yes Status filter for orders 0

Date Object

Attribute Type Description Sample
from string Start date (YYYY/MM/DD format) "2025/05/30"
to string End date (YYYY/MM/DD format) "2025/06/30"

Pagination Object

Attribute Type Description Sample
sortBy string Field to sort results by "id"
descending boolean Sort in descending order true
page integer Current page number 1
rowsPerPage integer Number of items per page 10
rowsNumber integer Total number of items 0
