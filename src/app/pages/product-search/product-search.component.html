POST /list

Product List Global Search

Retrieve a list of available products with filtering and pagination options. Token is not required but can be passed in the header for authenticated requests.

Request Attributes

Attribute Type Required Description Sample
search string No The search term for brand name, product name, etc. "bata"
filterBy string Yes The field to filter the search results by (e.g., "title") "title"
selectedCategories array No An array of selected category IDs to filter the search results [1,2,3]
pagination object Yes Pagination details for the search results See below

Pagination Object

Attribute Type Description Sample
sortBy string The field to sort the results by (e.g., "id") "id"
descending boolean Whether to sort the results in descending order false
page integer The current page number for paginated results 1
rowsPerPage integer The number of rows (items) to display per page 0
rowsNumber integer The total number of rows (items) available 0
